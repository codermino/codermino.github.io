    vuecli中指定模式mode
    2021/09/19 vue vuecli中指定模式mode

    使用mode

    vue-cli –mode 可以通过传递 –mode 选项参数为命令行覆写默认的模式

    Demo

    在vue cli项目中的package.json中进行如下配置

    1. test 指定模式为test –mode test
    2. production 指定模式为production –mode production
      {
  "scripts": {
    "serve": "vue-cli-service serve",
    "build": "vue-cli-service build",
    "lint": "vue-cli-service lint",
    "test": "vue-cli-service build --mode test",
    "production": "vue-cli-service build --mode production"
  }
}

    可以在配置公共的ip和端口的时候进行不同模式的判断

    const API_PREFIX = {
	USER: 'http://localhost:8210' //运维总线
};

// 测试环境
if (process.env.NODE_ENV === 'test') {
	API_PREFIX.USER = 'http://192.168.31.171:8210';
}

// 生产环境
if (process.env.NODE_ENV === 'production') {
	API_PREFIX.USER = 'http://192.168.31.167:8210';
}
