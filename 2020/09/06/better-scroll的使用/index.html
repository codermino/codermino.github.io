    滚动原理

    绿色部分为 wrapper，也就是父容器，它会有固定的高度。
黄色部分为 content，它是父容器的第一个子元素，它的高度会随着内容的大小而撑高。
那么，当 content 的高度不超过父容器的高度，是不能滚动的，而它一旦超过了父容器的高度，
我们就可以滚动内容区了。横向滚动的原理一致就是将固定高度改为固定宽度。（这里就不啰嗦了）

    纵向滚动

    <template>
  <div class="wrapper" ref="wrapper">
      <ul>
        <li v-for="item in 8">{{item}}</li>
      </ul>
  </div>
</template>
<script>
  import BScroll from 'better-scroll';
    export default {
      mounted() {
        this.$nextTick(() => {
          this.scroll = new BScroll(this.$refs.wrapper);
        });
      }
    };
</script>
<style type="text/css">
  .wrapper{
    overflow:hidden;
    height:100vh;
  }
  ul li{
    height:400px;
  }
</style>

    这是一个Vue BetterScroll纵向滚动demo,这里需要注意的有两点。

    • 只能有一层父级div，也就是容器
    • 父级div要设置溢出隐藏，并且固定高度

    横向滚动

    横向滚动，相比纵向滚动需要动态的去获取滚动区的宽度，直接上代码。

    <template>
  <div class="tab" ref="tab">

      created() {
        this.$nextTick(() => {
          this.InitTabScroll();
        });
      },
      methods:{
        InitTabScroll(){
          let width=0
          for (let  i = 0; i <this.itemList.length; i++) {
              width+=this.$refs.tabitem[0].getBoundingClientRect().width; //getBoundingClientRect() 返回元素的大小及其相对于视口的位置

                  startX:0,
                  click:true,
                  scrollX:true,
                  scrollY:false,
                  eventPassthrough:'vertical'
                });
              }else{
                this.scroll.refresh()
              }
          });
        }
      }
    };
  </script>

    横向滚动需要注意。

    • 只能有一层父级div，也就是容器
    • 父容器要设置溢出隐藏并固定宽度
    • 动态获取滚动区的宽度

    无法滚动的原因

    一、 DOM层级关系

    <div class="wrapper">
  <div class="content">
    content...
  </div>
</div>

    wrapper里面不能存在多个同级div，如果你这样写：

    <div class="classifyTitle"  ref="wrapper">
    <div class="">
	ss
    </div>
    <ul>
	<li v-for="(item,index) in classifyData.products">
	    <router-link :to="{name:'详情'}">{{item.title}}</router-link>
	</li>
    </ul>
</div>

    那么ul中的元素将不能滚动

    二、content是否被成功添加滚动相关style

    <div class="classifyTitle"  ref="wrapper">
	<ul>
	    <li v-for="(item,index) in classifyData.products">
		<router-link :to="{name:'详情'}">{{item.title}}</router-link>
	    </li>
	</ul>				
</div>

    封装better-scroll

    <template>
  <div ref="wrapper" :class="{wrapper:hasClass,wrapper2:!hasClass}">
    <div class="content">
      <slot></slot>
    </div>
  </div>
</template>

<script>
import BScroll from 'better-scroll'

export default {
  name: "Scroll",
  props:{
    // 当默认值是一个对象或者是数组的时候，default必须是一个函数
    probeType:{
      type:Number,
      default:0
    },
    pullUpLoad:{
      type:Boolean,
      default:false
    },
    hasClass:{
      type:Boolean,
      default:false
    }
  },
  data(){
    return{
      scroll:null
    }
  },
  mounted() {
    this.$nextTick(()=>{
      this.scroll=new BScroll(this.$refs.wrapper,{
        click:true,
        probeType:this.probeType,
        pullUpLoad:this.pullUpLoad
      });
      // console.log(this.scroll);
      // new BScroll().on()


      // 监听滚动事件
      if(this.probeType===2||this.probeType===3){
        this.scroll.on('scroll',(position)=>{
          // console.log(position);
          // 将scroll滚动的位置传递出去，因为真正使用的地方不是在这个组件中使用
          this.$emit('scroll',position);
        });
      }

      // 监听上拉事件
      if(this.pullUpLoad){
        this.scroll.on('pullingUp',()=>{
          // console.log('上拉');
          this.$emit('pullingUp');
        })
      }
    })

  },
  methods:{
    scrollTo(x,y,time=300){
      this.scroll&&this.scroll.scrollTo(x,y,time);
    },
    finishPullUp(){
      this.scroll&&this.scroll.finishPullUp();
    },
    refresh(){
      // console.log('图片加载');
      this.scroll&&this.scroll.refresh();
    },
    getScrollY(){
      // return this.scroll&&this.scroll.y;
      return this.scroll?this.scroll.y:0;
    }
  }
}
</script>

//一定要给wrapper一个滚动区域的高度并且overflow:hidden
