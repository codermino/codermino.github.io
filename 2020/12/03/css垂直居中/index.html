    垂直居中

    1. 单行文本垂直居中：
      如果一个容器中只有一行文字，对它实现居中相对比较简单，
      我们只需要设置它的实际高度height和所在行的高度line-height相等即可。
      div{
     height:25px; 
     line-height:25px;  
}
    2. 多行文本固定高度的居中（模拟table方法）
      css中垂直居中样式vertical-align:middle属性,
      但是此属性只对标签、、，
      和内联元素display设置为inline/inline-block起作用，其他的则不起作用
      (1)InternetExplorer6及以下的版本中是无效的vertical-align:middle
对表格起作用(可以用下面第4种方法)，那么可以用div来模拟成为table，
让vertical-align:middle属性起作用。
注意，display:table和display:table-cell的使用方法，前者必须设置在父元素上，后者必须设置在子元素上，
因此我们要为需要定位的文本再增加一个<div>元素：
      代码如下:
      div#wrap{
    height:400px; 
    display:table; 
}

div#content{ 
    vertical-align:middle;
    display:table-cell;
    border:1px solid #FF0099;
    background-color:#FFCCFF;
    width:760px;
} 

<body> 
<div id="wrap"> 
    <div id="content">
      <pre>现在我们要使这段文字垂直居中显示</pre></div>
    </div> 
</div>
</body>
    3. 使用用定位的方式position
      div#wrap{
    position:relative;
    overflow:hidden;
} 
div#content{ 
    position:absolute;
    top:50%; 
    height:400px;
    margin-top: -200px;
}
    4. 使用transition
      div#wrap{
    position: relative;
}
div#content{
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
}
    5. 使用flex
      div#wrap{
    display: flex;
    align-items: center;//垂直居中
    justify-content: center; //水平居中
}

    水平垂直居中

    1. div绝对定位水平垂直居中
      兼容性：,IE7及之前版本不支持
      div{
    width: 200px;
    height: 200px;
    background: green;
    position:absolute;
    left:0;
    top: 0;
    bottom: 0;
    right: 0;
    margin: auto;
}
    2. div绝对定位水平垂直居中
      div{
    width:200px;
    height: 200px;
    background:green;
    position: absolute;
    left:50%;
    top:50%;
    margin-left:-100px;
    margin-top:-100px;
}
    3. div绝对定位水平垂直居中
      兼容性：IE8不支持
      div{
    width: 200px;
    height: 200px;
    background: green;
    position:absolute;
    left:50%;    /* 定位父级的50% */
    top:50%;
    transform: translate(-50%,-50%); /*自己的50% */
}
    4. 弹性盒模型-css不定宽高水平垂直居中
      .box{
     height:600px;/去掉高度，只能垂直居中。
     display:flex;
     justify-content:center;
     align-items:center;
}
.box>div{
    background: green;
    width: 200px;
    height: 200px;
}
    5. 将父盒子设置为table-cell元素，可以使用text-align:center和vertical-align:middle实现水平、垂直居中。比较完美的解决方案是利用三层结构模拟父子结构
      <!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <div style="display: table-cell;vertical-align: middle; text-align: center; width: 300px;height: 300px;border: 1px solid #ccc;">
        <div style="display: inline-block;width: 200px;height: 200px;background-color: red;"></div>
    </div>
</body>
</html>
    6. 对子盒子实现绝对定位，利用calc计算位置
      <p class="outerBox calc">
    </p><p class="innerBox">calc</p>
<p></p>
/*绝对定位，clac计算位置*/
.calc{
  position: relative;
}
.calc .innerBox{
  position: absolute;
  left:-webkit-calc((500px - 200px)/2);
  top:-webkit-calc((120px - 50px)/2);
  left:-moz-calc((500px - 200px)/2);
  top:-moz-calc((120px - 50px)/2);
  left:calc((500px - 200px)/2);
  top:calc((120px - 50px)/2);
}
    7. display:table
      <!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8" />
		<title></title>
		<style>
			.con{
				background: red;
				display: table; /* 将父元素div模拟为块级表格 table*/
				margin: 50px;   /* 设置父元素 table 的margin*/
			}
			.container{
				width: 200px;
				height:50px;
				border: 1px solid black;
				display: table-cell;	 	
				vertical-align: middle; 
				text-align: center;		
				/*margin: 50px;*/  去掉设置无效的 margin
			}
		</style>
	</head>
	<body>
		<div class="con">
			<div class="container">
			a
			</div>
			<div class="container">
				b
			</div>
			<div class="container">
				c
			</div>
		</div>
		
	</body>
</html>
