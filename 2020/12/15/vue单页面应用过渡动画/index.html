    Demo1(使用animate.css)

    引入

    @import './css/animate.css';

    使用

    <transition
  name="page"
  mode="out-in"
  enter-active-class="animated fadeIn"
  leave-active-class="animated fadeOut"
>
  <router-view></router-view>
</transition>

    Demo2(使用原生js)

    html部分

    <transition name="fade">
  <router-view></router-view>
</transition>

    css部分

    .fade-enter,
  .fade-leave-to{
    visibility: hidden;
    opacity: 0;
  }
  .fade-leave-to{
    display: none;
  }
  .fade-enter-active,
  .fade-leave-active{
    transition: all .3s ease;
  }
  .fade-enter-to,
  .fade-leave{
    visibility: visible;
    opacity: 1;
  }

    mode

    的默认行为 - 进入和离开同时发生。
    同时生效的进入和离开的过渡不能满足所有要求，所以 Vue 提供了过渡模式

    • in-out：新元素先进行过渡，完成之后当前元素过渡离开。
    • out-in：当前元素先进行过渡，完成之后新元素过渡进入。
