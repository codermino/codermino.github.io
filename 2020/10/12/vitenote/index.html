    2020/10/12 vite vite笔记

    vite的优点

    1. 快如闪电的轻量级的冷服务启动
    2. 瞬间的热更新
    3. 真正的随机应变的编译
    4. 能够了解更多的细节和为什么

    开始

    note:目前vite只工作在vue3的版本上，也就是说vue2的版本是无法使用的

    $ npm init vite-app <project-name>
$ cd <project-name>
$ npm install
$ npm run dev

    对编译的支持

    TypeScript

    Vite supports importing .ts files and <script lang="ts"> in Vue SFCs out of the box.

    CSS / JSON Importing

    官网介绍

    You can directly import .css and .json files from JavaScript (including <script> tags of *.vue files, of course).
* .json files export their content as an object that is the default export.
* .css files do not export anything unless it ends with .module.css (See CSS Modules below). Importing them leads to the side effect of them being injected to the page during dev, and being included in the final style.css of the production build.
Both CSS and JSON imports also support Hot Module Replacement.
    1. vite对于css和json移入都是支持的
    2. .json文件内容作为对象导出
    3. .css文件不需要导出，除非它以.module.css结尾
    4. CSS和JSON引入也支持热更新

    使用xxx.module.css的方式引入

    son.module.css文件内容

    .son2 .title{
    color: green;
}

    引入

    import sonstyle from './css/son.module.css'

composition API中使用并return
setup(props,context){
    return{
      sonstyle
    }
}

    使用

    <div class="son" :class="sonstyle.son2">
    <h1>123</h1>
    <h2 :class="sonstyle.title">456</h2>
</div>

    在.vue文件中使用

    style module的方式引入

    <style scoped vars="{ color }" module>
  .text {
    color: var(--color);
  }
</style>

    使用

    <h3 :class="$style.text">{{people.name}}-{{people.id}}</h3>

    Dev Server Proxy

    const path = require('path')

module.exports = {
  proxy: {
    '/api': {
      target: 'http://apis.juhe.cn',
      changeOrigin: true,
      rewrite: path => path.replace(/^\/api/, '')
    }
  },
  alias: {
    // 键必须以斜线开始和结束
    '/@/': path.resolve(__dirname, './src')
    // '/@components/': path.resolve(__dirname, './src/components')
  }
}

    alias(设置别名)

    设置

    alias: {
    // 键必须以斜线开始和结束
    '/@/': path.resolve(__dirname, './src')
    // '/@components/': path.resolve(__dirname, './src/components')
  }

    使用

    import sonCom from '/@/components/sonCom.vue'
