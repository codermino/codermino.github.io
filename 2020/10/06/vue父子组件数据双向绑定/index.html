    Vue单向数据流

    单向数据流：父组件可以向子组件通过属性形式传递参数，
    传递的参数也可以随时随意修改；
    但子组件不能修改父组件传递过来的参数。
    只能用父组件传递的数据。

    原因

    原因：因为子组件接收父组件传递过来的数据，并不是一个基础类型的数据类型，
而是类似于object这种引用类型的数据类型。如果修改了引用类型的数据，
在使用子组件时，则会造成其他子组件接收数据的错误

    方法一

    父组件
<child-test :inputData="inputData" @inputdataChange="inputdataChange"></child-test>
<span>{{inputData}}</span>

子组件
<template>
  <div class="childTest">
    输入内容<input type="text" :value="inputData" @input="inputChange">
  </div>
</template>

<script>
export default {
  name: "childTest",
  props: {
    inputData: {
      type: String,
      default: ''
    },
  },
  methods: {
    inputChange(e) {
      this.$emit('inputdataChange',e.target.value)
    }
  },
}
</script>

    方法二(使用v-model)

    官方介绍

    允许一个自定义组件在使用 v-model 时定制 prop 和 event。默认情况下，
**一个组件上的 v-model 会把 value 用作 prop 且把 input 用作 event**
但是一些输入类型比如单选框和复选框按钮可能想使用 value prop 来达到不同的目的。
使用 model 选项可以回避这些情况产生的冲突。

    父组件不需要做方法监听，而是通过v-model指令绑定一个变量即可。
    1、子组件通过props接受一个父组件通过v-model指令绑定的变量

    props: ['value']

    2、子组件新建一个model对象

    model: {
  prop: 'value',  // props接受的变量名称
  event: 'change'  //定义一个方法
}
    父组件
<child-test2 v-model="inputData"></child-test2>
{{inputData}}
父组件则只要v-model绑定一个变量即可，当输入框改变时inputValue就会得到响应。
子组件
<template>
  <div class="childTest2">
<!--    输入值: <input type="text" :value="value" @input="inputChange">-->
<!--    输入值: <input type="text" :value="inputData" @change="inputValueChange">-->
    输入值: <input type="text" :value="inputData" @input="inputValueChange">
  </div>
</template>

<script>
export default {
  name: "childTest2",
  // props: {
  //   value: {
  //     type: String
  //   }
  // },
  props: {
    inputData: {
      type: String
    }
  },
  model: {
    prop: 'inputData',  // props接受的变量名称
    event: 'change'  //定义一个方法(默认为input)
  },
  methods: {
    // inputChange(e) {
    //   this.$emit('input',e.target.value);
    // },
    inputValueChange(e){
      this.$emit('change',e.target.value);
    }
  },
}
在input框改变的事件里通过$emit触发在model对象里面定义的change方法,并传入输入的的值。
</script>

    Demo

    父组件

    <template>
  <div>
    <model-test-son v-model="msg"></model-test-son>
    {{msg}}
  </div>
</template>

<script>
import modelTestSon from './modelTestSon'
export default {
  name: "modelTest",
  components:{
    modelTestSon
  },
  data() {
    return {
      msg: '123'
    }
  },
}
</script>

    子组件

    <template>
<div @click="changeMsg">点击改变msg</div>
</template>

<script>
export default {
  name: "modelTestSon",
  model: {
    prop: 'msg',  // props接受的变量名称
    event: 'change'  //定义一个方法(默认为input)
  },
  props: {
    msg: {
      type: String,
      default:''
    }
  },
  methods: {
    changeMsg() {
      this.$emit('change', '456')
    }
  },
}
</script>
